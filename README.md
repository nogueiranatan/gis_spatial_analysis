# Nate’s GIS Spatial Analysis Portfolio

Welcome to my GIS portfolio showcasing geospatial analysis projects using QGIS, Excel, and geospatial datasets. This collection demonstrates skills in critical reasoning, spatial data processing, mapping, and strategic cartographic communication.

---

## 📌 Projects

### 🔥 Gold Coast Hinterland Bushfire Vulnerability Assessment
A GIS-based vulnerability model assessing how landscape characteristics influence bushfire vulnerability across the Gold Coast Hinterland. The project integrates vegetation fuel, slope, historical fire behaviour, and fire recurrence into a standardised spatial decision-support framework.

**Tools:** QGIS  
**Key methods:** Multi-Criteria Evaluation (MCE), raster analysis, spatial aggregation, attribute joins, reclassification, weighted overlay modelling, vulnerability mapping, cartographic communication

- [View Project](Bushfire_vulnerability/README.md)

---

### 🇺🇸 US Presidential Election - Bivariate Choropleth Analysis
A county-level electoral map visualising both the winning political party and the strength of each county’s majority vote using expression-driven opacity.

**Tools:** QGIS, Excel  
**Key methods:** Attribute joins, Expression Builder, categorized symbology, expression-driven opacity, custom legend design, cartographic communication

- [View Project](US_election_map/README.md)

### 🗺️ US Median Household Income - Choropleth Analysis
A county-level analysis of median household income across the United States using QGIS.

**Tools:** QGIS, Excel  
**Key methods:** Attribute joins, choropleth classification (Natural Breaks), cartographic design  

- [View Project](US_income_choropleth/README.md)

### 🇬🇧 UK General Electoral Map - Categorised Symbology & Callout Annotation

**Tools:** QGIS  
**Key methods:** Categorized Symbology (party classification), Filters (London region), Sorting (descending seats won), Inset map, Map Overview, Callout Annotation (linking London region to inset map)
